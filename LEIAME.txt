				
			LISTAS

---------------------------------------------

!!!	Objetivo do programa Trab2-1: 
	
	Testar o módulo de listas genéricas. 
	Testar o módulo de listas genéricas. 

---------------------------------------------

!!! Modo de uso do programa
	
	Abra a pasta Trab2-1.
	
	Clicar no arquivo Lista.bat.
	
	O programa será executado gerando um arquivo LogLista.txt onde poderão ser verificados os resultados dos testes feitos pelo arquivo TesteLista.script.
 
	Caso o usuário deseje criar seu próprio script de teste, pode-se optar por 2 formas distintas de fazê-lo executar. Para isso, deve:
		
	1 - Alterar o nome do script que o usuário criou para TesteLista.script e então clicar duas vezes no arquivo Lista.bat.
	2 - Ou, clicar com o lado direito do mouse no arquivo Lista.bat e selecionar a opção Edit (Editar) para alterar o conteúdo do arquivo. 
	Alterar o trecho onde está o nome do arquivo contendo o script de teste, atualmente TesteLista.script, para o nome do seu script. 


---------------------------------------------

!!! Descrição do conteúdo do arquivo Lista.bat

  Trab2-1 /sTesteLista.script >logLista.txt

  <TesteLista.script>  		- nome do arquivo contendo o script de teste.
							Parâmetro obrigatório, exceto se solicitado auxílio.
  <logLista.txt>      		- nome do arquivo contendo o script de teste.
							Parâmetro opcional. Se não fornecido exibe
							o resultado do teste na tela (stdout)
